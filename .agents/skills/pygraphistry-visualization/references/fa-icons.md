# Font Awesome Icon Guidance (PyGraphistry)

Use Font Awesome icon names (without the `fa-` prefix) for Graphistry icon encodings.

Examples:
- `user`
- `building`
- `triangle-exclamation`
- `circle-info`
- `magnifying-glass`

Notes:
- Prefer stable Free icon names for portability.
- Validate icon names against the official catalog before finalizing.
- Keep fallback mappings for unknown categories when possible.

References:
- Font Awesome Free icon search: https://fontawesome.com/search?o=r&m=free
- PyGraphistry icon encoding docs: https://pygraphistry.readthedocs.io/en/latest/api/plotter.html
