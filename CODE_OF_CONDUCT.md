# Graphistry Skills Code of Conduct

This project is maintained as a respectful, collaborative engineering space.

## Expected Behavior

- Be constructive and professional.
- Focus on ideas, tradeoffs, and outcomes, not personal attacks.
- Help newcomers with clear, actionable guidance.

## Unacceptable Behavior

- Harassment, intimidation, or hostile conduct.
- Discrimination or exclusionary language.
- Sharing private information without consent.

## Scope

This applies to all project spaces, including GitHub issues, pull requests, discussions, and direct project communications.

## Reporting

Report violations to `oss@graphistry.com`. We will review confidentially and take appropriate action, including warnings or removal from project participation.
